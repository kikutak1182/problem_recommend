Atcoderのユーザーネームと学びたいアルゴリズムを入力すると、関連した自分のレートに近いdifficultyの問題を提案してくれます。

sentencetransformerで入力アルゴリズムとタグとのコサイン類似度を計算し、類似度の高いタグを抽出、関連する問題を検索してユーザーのレートに近い問題を推薦します。
複数のアルゴリズム/単語を組み合わせた検索が可能です。
